#Soy Functions.

Additional functions for Closure Templates a.k.a. Soy.

To setup and build, run:
```
$ ./run.sh setup && ./run.sh build
```


To test:
```
$ ./run.sh test
```


To create JavaScript templates example soy file:
```
$ ./run.sh example-js
```
